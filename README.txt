Plugin allows easily add JavaScript, HTML or CSS code to your web pages.

== Description ==

Code Inserter is an efficient WordPress plugin that allows you to customize your website’s appearance and functionality. Would like to add CSS, JavaScript or HTML code? With our plugin you can build a page which will meet all your requirements.

What’s in it for you?
Code inserted was designed to be user experience and be simple in use. All you need is just our plugin and the idea which code area you would like to customize.

Our plugin provides the opportunity to be used in the following code areas:
* Head section. Code will be <head>here</head>
* Body section. Code will be <body>here</body>
* Code after h1 in the pages, posts, terms. Code will be <h1>code</h1>here
* Footer section. Your code will be <footer>here</footer>
* Content section. Code will be added in your content section.

It's just a small piece of functionality that the Code Inserter supplies. Except this, you will be also able to:
* Choose the specific area, where the plugin will be used. For example: after <head> or before </head>
* Insert code also on the AMP version of your pages (if it exists).
* Work with the plugin even if your site has a multisite structure.
* Easily add the Google Tag Manager. We know that SEO is very important for you all.

Sounds great?
Download our plugin and start to work with it right now. It's absolutely free!

== Installation ==

1. Upload `code-inserter` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
